# foodfinder

프로젝트 소개 : 이 프로젝트는 사용자들이 음식점을 탐색하고 리뷰를 작성하며, 정보를 공유할 수 있는 웹 프로젝트입니다. 
React를 기반으로 개발되었으며, 주요 기능으로 음식점 검색, 상세 정보 제공, 리뷰 작성 및 열람, 로그인 등이 있습니다. 
특히, 카카오맵 API를 활용한 지도 관련 기능에 중점을 두고 있습니다.
